# IDEAL COLLOR - Sistema de Ordem de Serviço (com Supabase)

Projeto em React + Vite integrado ao Supabase para gestão de clientes da IDEAL COLLOR.
Inclui:
- Login (Supabase Auth)
- Proteção de rotas
- Cadastro e listagem de clientes (tabela `clients`)

Para rodar:
1. npm install
2. Criar arquivo .env.local com VITE_SUPABASE_URL e VITE_SUPABASE_ANON_KEY
3. npm run dev
